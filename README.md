# SpringFramework_Airport
스프링 프레임워크를 이용한 항공 예약 홈페이지

### 0. 개요
------------------------------
1. 프로젝트 정의
여러 항공권의 정보를 한눈에 볼 수 있도록 하여 사용자가 간편하게 항공기 운행 목록을 조회하고, 좌석을 선택하여 예약할 수 있는 프로그램을 만들고자 한다.

2. 프로젝트 기능
항공권 운행정보 조회
항공권 좌석정보 조회
항공권 예약 및 예약취소
사용자 정보 조회 및 수정
